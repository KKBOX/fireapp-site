Using CoffeeScript

Fire.app has built-in CoffeeScript support. With Fire.app, you can use CoffeeScript easily:

  1. Add the folder coffeescripts/ in your project folder.
  2. Add a file in the folder coffeescripts/ and name it with corresponding file extension .js.coffee , then you can start to write CoffeeScript. For example: app.js.coffee.
  3. Files in the coffeescripts/ folder are compiled to javascript in the folder javascripts/ automatically after saving. To load this external file in the HTML, just insert a javascript statement like <script src="/javascripts/app.js"></script>.

If you want to modify the default folder name, please click "Change Options" and change the folder name, then click "Save".
