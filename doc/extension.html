Using Extensions

We have put some extensions into Fire.app, and there are two ways to add more 3rd party extensions.

Adding with Fire.app

Click the Fire.app icon and select "Open Extensions Folder." Fire.app will open the specific folder where you can put extensions. The extensions inside this folder are automatically loaded when Fire.app starts. These extensions can be use in any project being watched by Fire.app. The extensions are also available from the "Create Project" menu.

Besides adding new extensions to Fire.app, you can overwrite any existing extensions with the same method.

Example: Install Fancy Buttons

Here we use installing fancy-buttons as our example.

  1. Download the fancy-buttons zip file
  2. Unzip the zip file.
  3. Open the "lib" folder, you will see a file used to register paths. In this case, the file name is "fancy-buttons.rb".
    IMPORTANT: The extension's folder name must be renamed as this file name, in this case, "fancy-buttons."
  4. Click the Fire.app icon and select "Open Extensions Folder," you'll see the extensions folder open.
  5. Move the renamed folder to the extensions folder.
  6. Quit Fire.app and start it again. You'll see the extension appear in the "Create Project" menu.

Adding with Ruby gems

WARNING: This is for advanced users only.

You can choose "Compass Version: Custom" in preferences, and specify your own gem paths. Fire.app will try to load it. Since Fire.app uses JRuby, native gems will not work.
